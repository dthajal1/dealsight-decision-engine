# Deal Verdict Engine

AI-powered M&A deal analysis platform. Upload documents, run automated due diligence, and get structured verdicts across every stage of a deal.

## Features

- **Pipeline View** — Kanban board to track deals across stages
- **CIM Analysis** — Automated Confidential Information Memorandum breakdown
- **Financial Screening** — Key metrics extraction and risk flagging
- **LOI Builder** — Letter of Intent anchor analysis
- **Quality of Earnings (QoE)** — Upload QuickBooks CSVs for financial due diligence via n8n webhook
- **Agreement Review** — Close-stage document analysis

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (database, storage, edge functions)
- n8n (webhook-based AI analysis pipelines)
